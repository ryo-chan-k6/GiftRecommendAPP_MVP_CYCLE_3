# 1. 正本定義の前提

## ■ 定義（重要）

```
正本 = そのデータの「最終的な真実」を保持する唯一の場所
```

---

## ■ 判断基準

| 観点                       | 正本にするか |
| -------------------------- | ------------ |
| 再計算できる               | ❌           |
| ログから復元できる         | ❌           |
| 外部から取得可能           | ❌           |
| それが無いと業務成立しない | ✅           |
| 履歴・再現に必須           | ✅           |

---
